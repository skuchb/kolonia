# KOLONIA design mock

Statyczny, minimalistyczny mock strony dla daily guessing game w uniwersum
Gothic 1. Ten etap nie zawiera logiki rozgrywki, storage, API ani walidacji
odpowiedzi. Widoki są ręcznie wypełnione przykładowymi stanami, żeby ocenić
kierunek graficzny.

## Uruchomienie

```bash
npm install
npm run dev
```

Lokalny podgląd: `http://localhost:3000/`.

## Główne pliki

- `app/page.tsx` - statyczna struktura mocku.
- `app/globals.css` - cała warstwa wizualna i responsywność.
